# Laravel Bookstore CRUD

## Model

- Category
- Publisher
- Author
- Book

## Installation
- Install PHP
- Install Composer : [Download](https://web.archive.org/web/20170425173028/https://getcomposer.org/download/)
- Install laravel  
  ```composer global require “laravel/installer”```
